Debounce Example

DIRECTIONS: Try to resize this browser window.

PURPOSE: We are trying to trigger an update in number of times the user FINISHES resizing the browser window. Here we are not interested in having the count increase as the user is dragging the window - which would be the NORMAL behavior.

Here the function will be called after a delay (650 milliseconds) of no resizing the window ONCE RESIZING the window has started.

The # of times that the window was finished resizing: 0
